Q: What is the AI Chatbot project developed by kashaf naveed?

Q: What technologies are used in the AI Chatbot?

## Try-On Spectacles (MERN + AI)

Q: What is the Try-On Spectacles project?
A: It is a real-time virtual try-on application that allows users to preview eyewear digitally across devices.

Q: Which AI technologies are used in the Try-On Spectacles system?
A: The system uses MediaPipe for facial landmark detection, ONNX runtime for optimized inference, and OpenCV for image processing.

Q: What backend and frontend technologies were used?
A: The backend is built with FastAPI for AI inference, and the frontend uses the MERN stack for a seamless user experience.

Q: What problem does this project solve?
A: It reduces the need for physical try-ons by providing an accurate, low-latency digital alternative.

## Music Genre Converter (AI Project)

Q: What is the Music Genre Converter project?
A: It is an AI system that converts music from one genre to another using deep learning techniques.

Q: Which model is used for music transformation?
A: A Variational Autoencoder (VAE) is used to learn latent representations of musical styles.

Q: How is audio processed in this project?
A: Audio is converted into spectrograms, which are used as input for model training and inference.

Q: What is the main purpose of this project?
A: To explore creative applications of AI in audio processing and demonstrate deep learning beyond text data.
